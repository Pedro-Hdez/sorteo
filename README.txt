Para correr la aplicación:

1.- Instalar los requerimientos del archivo "sorteo_terrenos.yml" (Entorno de Anaconda).

2.- Añadir los archivos 'participantes.csv' y 'terrenos.csv' a la carpeta "datos/" (Ya se encuentran ahí los dos csv's más recientes)

3.- Ejecutar el script 'main.py' con python (python main.py)

** El script 'sorteo.py' contiene las funciones con las que se realiza el sorteo aleatorio de boletos y terrenos. **
